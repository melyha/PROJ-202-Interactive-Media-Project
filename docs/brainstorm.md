# 🎮 Enigma — Brainstorming & Research
> **Course:** Interactive Media Project · Saskatchewan Polytechnic · 2026  


---

## Section 1 — Brainstorming

Three interactive media project ideas were explored, each targeting a different format, technology stack, and engagement model. The goal was to identify a concept that is creative, feasible within a 4-week timeline, and personally motivating to build.

---

###  Idea 01 — 2D Web Platformer Game

| Field | Details |
|---|---|
| **Purpose** | Entertain and showcase game development skills. Players explore a mysterious pixel-art world, collecting items and navigating obstacles to reach the end of each level. |
| **Target Audience** | Casual gamers aged 15–30 who enjoy retro-style browser games. No installation required, playable instantly in any browser. |
| **Interactivity** | Keyboard-driven (WASD / arrow keys). Physics-based movement: gravity, jump arcs, collision. Players interact with platforms, collectibles, and enemies in real time. |
| **Format & Stack** | Browser-based web game built with Phaser.js 3. Hosted on GitHub Pages. Pixel art assets sourced from itch.io free packs. |

---

###  Idea 02 — Word / Math Puzzle Game

| Field | Details |
|---|---|
|  **Purpose** | Educate and entertain through bite-sized logic challenges: a math equation guesser, a typing speed game, or a word-unscramble mechanic. All achievable within days. |
|  **Target Audience** | Students and puzzle enthusiasts aged 12–35. Works well as a casual daily-challenge app. |
|  **Interactivity** | Tap/click-based or keyboard input. Question → answer → feedback loop. Visual cues (colour, shake, glow) for correct/wrong responses. Optional timer creates pressure. |
|  **Format & Stack** | Web app using React + Vite or plain HTML/CSS/JS. Or React Native + Expo for an Android APK. |

---

###  Idea 03 — AR Mobile Game (Play Store)

| Field | Details |
|---|---|
|  **Purpose** | Blend the physical and digital worlds through augmented reality gameplay. Players point their phone camera at real surfaces to spawn game objects, enemies, or interactive scenes. |
|  **Target Audience** | Mobile-first users aged 16–35 interested in AR/location-based play. Inspired by Pokémon GO's mainstream AR success, smaller in scope, more creative in direction. |
|  **Interactivity** | Camera-based AR interaction using the phone's real-world environment as the play surface. Touch to interact with AR objects, device tilt for navigation. |
|  **Format & Stack** | React Native + ViroReact or Unity + AR Foundation. Targets Android (Play Store). Requires ARCore support. Higher complexity but strongest novelty factor. |

---

### 🗳️ Selected Concept

> **Idea 01 — 2D Web Platformer** was selected for its clear technical scope, strong learning value, and realistic fit within a 4-week development timeline using Phaser.js.
